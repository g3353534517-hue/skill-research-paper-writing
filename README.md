# Research Paper Writing / 学术论文写作流程

![分类](https://img.shields.io/badge/分类-写作-green)

## 项目简介

学术论文写作的详细流程规范技能。与 research-paper-writer 互补，侧重写作流程管理而非内容生成。

## 流程规范

1. 确定研究问题和假设
2. 文献检索与整理
3. 拟定论文大纲
4. 分章节撰写
5. 内部审阅与修改
6. 格式终检与提交

## 与 research-paper-writer 的区别

- research-paper-writer：侧重内容生成和质量
- 本技能：侧重流程管理和进度控制
